# QA Pet Project — Manual & API Testing

Проект включает **ручное тестирование веб-приложения** и **автоматизацию API-тестов**

## Проект 1: Тестирование nopCommerce
Проведено функциональное тестирование демо-версии интернет-магазина **NopCommerce**

### Что сделано:
- CheckList
- Test Cases
- Bug Reports

### Область тестирования
- Регистрация и авторизация
- Каталог товаров
- Корзина
- Оформление заказа

### Инструменты
- Jira
- Test IT
- Google Sheets

## Проект 2: API тестирование
Тестирование REST API сервиса **go-rest.io**

### Инструменты
- Postman
- Python
- Библиотеки Pytest, Requests

### Что сделано
- Коллекция Postman (CRUD операции)
- Автоматизированные API тесты на Pytest
- Проверка статус-кодов и тела ответов
